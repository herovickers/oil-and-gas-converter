## How To Use
     
From your terminal:

```bash
# Clone this repository
$ git clone https://github.com/herovickers/oil_and_gas_unit_converter.git

# Go into the repository
$ cd oil_and_gas_unit_converter
```
Then
```bash
# Run the app
$ flutter run
```
